# ShrinkIt
Potentially the world's worst URL shortener in existence

## How does it work?
Add links and their aliases to a csv, run it through `flask run` or gunicorn if in prod, boom there you go.

## Why a CSV and not an actual database?
I made this in like an hour, i also did not want to learn sql or whatever while half asleep.
